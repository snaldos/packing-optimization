Detailed Description

Implements brute-force and backtracking algorithms for the 0/1 Knapsack Problem.

  • Brute-force (explicit subset generation): O(2^n * n) time, O(n) space for best solution.
  • Backtracking: O(2^n) time in worst case, but can be faster with pruning; O(n) space for recursion and best solution.
