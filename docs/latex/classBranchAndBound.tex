\doxysubsection{Detailed Description}
Implements the branch-\/and-\/bound algorithm for the 0/1 Knapsack Problem. 

Uses upper bound estimation (fractional knapsack) to prune the search space.

Time complexity\+: O(2\texorpdfstring{$^\wedge$}{\string^}n) in the worst case, but typically much faster due to pruning. Space complexity\+: O(n) for recursion and solution vectors.
