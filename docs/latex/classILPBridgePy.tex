\doxysubsection{Detailed Description}
Bridges C++ and Python to solve the 0/1 Knapsack Problem using ILP (via Pu\+LP). 

Serializes the problem to JSON, calls a Python script to solve it using Pu\+LP, and parses the result.

Time complexity\+: Depends on the Python ILP solver; exponential in the worst case, but often efficient for moderate n. Space complexity\+: Depends on the solver\textquotesingle{}s internal representation (typically O(n)).
