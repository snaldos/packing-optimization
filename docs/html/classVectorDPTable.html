Detailed Description

Concrete DP table implementation using a 2D vector for bottom-up DP.

Stores subproblem solutions in a dense (n+1) x (max_weight+1) table. Used for classic bottom-up dynamic programming approaches to knapsack.

Time complexity for access/set: O(1) Space complexity: O(nW)
