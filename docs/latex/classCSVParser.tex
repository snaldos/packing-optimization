\doxysubsection{Detailed Description}
Parses CSV files to extract pallet and truck data. 

Provides methods to read and parse CSV files containing pallet and truck information, populating vectors of \doxylink{classPallet}{Pallet} objects and \doxylink{classTruck}{Truck} objects for use in algorithms.
