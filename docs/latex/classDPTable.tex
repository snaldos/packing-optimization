\doxysubsection{Detailed Description}
Abstract interface for dynamic programming tables used in knapsack algorithms. 

Provides a common interface for both vector-\/based and hashmap-\/based DP tables. Used to store and retrieve subproblem solutions for dynamic programming approaches.
