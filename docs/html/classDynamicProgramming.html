Detailed Description

Implements dynamic programming algorithms for the 0/1 Knapsack Problem.

Supports both bottom-up (vector table) and top-down (hashmap memoization) approaches.

  • DP Vector: O(nW) time, O(nW) space (can reconstruct solution).
  • DP HashMap: O(s) time, O(s) space, where s is the number of unique (i, w) states (sparse for some datasets).
  • DP Optimized (2 rows): O(nW) time, O(W) space (only computes max profit, no reconstruction).
