\doxysubsection{Detailed Description}
Implements dynamic programming algorithms for the 0/1 Knapsack Problem. 

Supports both bottom-\/up (vector table) and top-\/down (hashmap memoization) approaches.


\begin{DoxyItemize}
\item DP Vector\+: O(n\+W) time, O(n\+W) space (can reconstruct solution).
\item DP Hash\+Map\+: O(s) time, O(s) space, where s is the number of unique (i, w) states (sparse for some datasets).
\item DP Optimized (2 rows)\+: O(n\+W) time, O(\+W) space (only computes max profit, no reconstruction). 
\end{DoxyItemize}
