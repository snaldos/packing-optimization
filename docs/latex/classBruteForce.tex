\doxysubsection{Detailed Description}
Implements brute-\/force and backtracking algorithms for the 0/1 Knapsack Problem. 


\begin{DoxyItemize}
\item Brute-\/force (explicit subset generation)\+: O(2\texorpdfstring{$^\wedge$}{\string^}n \texorpdfstring{$\ast$}{*} n) time, O(n) space for best solution.
\item Backtracking\+: O(2\texorpdfstring{$^\wedge$}{\string^}n) time in worst case, but can be faster with pruning; O(n) space for recursion and best solution. 
\end{DoxyItemize}
