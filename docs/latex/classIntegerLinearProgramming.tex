\doxysubsection{Detailed Description}
Solves the 0/1 Knapsack Problem using Integer Linear Programming (ILP). 

Formulates the problem as an ILP and solves it using an external solver (e.\+g., OR-\/\+Tools CBC).

Time complexity\+: Depends on the ILP solver; exponential in the worst case, but often efficient for moderate n. Space complexity\+: Depends on the solver\textquotesingle{}s internal representation (typically O(n)).
