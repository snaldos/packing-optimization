\doxysubsection{Detailed Description}
Implements a greedy approximation algorithm for the 0/1 Knapsack Problem. 

Selects pallets based on their profit-\/to-\/weight ratio (not fractional knapsack).

Time complexity\+: O(n log n) (for sorting) + O(n) (selection) = O(n log n) Space complexity\+: O(n) (for sorted copy and output)
